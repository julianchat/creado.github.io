La etiqueta video: para especificar la ubicacion de el archivo que se reproducira, puede usar el elemento <source> dentro del elemento <video>. (esto debe reemplazar el atributo src en la etiqueta <video> de apertura.) Tambien puede utilizar varios elementos <source> para especificar que el video esta disponible en varios formatos. (debido a un erros en el ipad, debe proporcionar el video mp4 como el primer formato. De lo contrario, puede que no inicie.

src:este atributo especifica la ruta de acceso al video.

type:se debe utilizar este atributo para decirle al navegador que formato es el video de lo contrario, descargara parte del video para ver si puede reproducir el archivo (que tomara tiempo y ancho de banda).

codecs:el codec que uso para codificar el video se suministra dentro de el atributo type tenga en cuenta el uso de las comillas simples, asi como comillas dobles en el atributo de tipo, cuando se suministra
