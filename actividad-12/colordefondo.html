Background Color - Color de Fondo

Si no no se especifica un color de fondo es transparente.

CSS trata cada elemento HTML como un cuadro y la propiedad backgroung-color establece el color del fondo para el cuadro. Se ouede especificar un color de fondo de las mismas tres formas en que puede especificar colores de Primer plano: Valores RGB, Codigo Hexadecimal y nombre de colores. Tambien hemos utilizado la propiedad padding para separar el texto de los bordes de los cuadros esto hace que sea mas facil de leer
