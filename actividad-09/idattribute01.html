cada elemnto de HTML puede llevar el atributo id. se utiliza para identificar de forma unica ese elemto de otros elementos de la pagina. Su valor debe comenzar con una letra o un guion bajo (no un numero o cualquier otro caracter). Es importante que no hay dos elementos en la misma pagina tienen el mismo valor para sus atributos ID(de lo contrario, el valor ya no es unico)

dar a un elemento una identidad unica le permite darle un estilo diferente que cualquier otra instacia del mismo elemento en la pagina.

por ejemplo, es posible que desee asignar un parrafo dentro de la pagina (quizas un parrafo que contiene una cotizacion de una extraccion) un estilo diferente que todos los demas parrafos. en el ejemplo, el parrafo con el atributo ID cuyo valor es pullquote se hace en mayusculas usando CSS.
